# 🎬 Netflix Data Analysis – Internship Task 1

This project is part of my **Data Analysis Internship at CODTECH**.  
It focuses on analyzing Netflix's Movies and TV Shows dataset using **Python and pandas**.

---

## 📌 What This Project Does

- Reads a real Netflix dataset (.csv format)
- Displays basic info about the data
- Shows total number of Movies vs TV Shows
- Lists top 10 countries with most content
- Lists top 10 most common genres
- Shows how much content was added each year
